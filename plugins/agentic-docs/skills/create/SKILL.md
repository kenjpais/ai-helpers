# Agentic-Docs: Create

**Trigger**: `/agentic-docs:create`  
**Purpose**: Generate complete agentic documentation following the Agent Knowledge Framework

## Overview

This skill generates comprehensive agentic documentation by analyzing repository files directly. Each documentation file is generated by a dedicated skill/prompt for modularity and clarity.

**Key Principles:**
- **No external ingestion**: Repository files are the sole input source
- **File-specific generation**: Each output file has dedicated generation logic
- **Full observability**: Log all data sources, tools, and skills used
- **Timestamped logs**: Every run produces CLI output + log file

## Input

**Repository Path** (optional - defaults to current directory)

```
/agentic-docs:create [<repo-path>]
```

**Examples**:
```bash
/agentic-docs:create                           # Current directory
/agentic-docs:create /path/to/openshift/installer
/agentic-docs:create ~/repos/kubernetes-operator
```
